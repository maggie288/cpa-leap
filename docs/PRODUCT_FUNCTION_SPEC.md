# CPA Leap 产品功能设计文档（V1）

## 1. 产品目标

打造一个面向 CPA 备考用户的 H5 学习产品，借鉴多邻国的课程节奏与激励机制，通过大语言模型生成「结构化课程 + 训练题 + 复习建议」，帮助用户低门槛持续学习并提升做题正确率。

## 2. 多邻国课程设计研究（可迁移要点）

基于公开研究与官方材料，抽象出可直接迁移到 CPA 学习的机制：

1. **短时高频**：每课 5-15 分钟，降低启动成本。
2. **路径式解锁**：单元/课时按顺序推进，形成“下一个就学它”的清晰引导。
3. **即时反馈**：做完立即判分、解释、错因归类。
4. **个性化难度**：根据错题与掌握情况调整题目难度和知识点重心。
5. **间隔复习**：将旧错题按时间窗口回推复现，强化长期记忆。
6. **游戏化激励**：XP、连学天数、进度条、阶段目标，驱动留存。

## 3. 用户与场景

- **核心用户**：在职 CPA 备考生（碎片化学习，时间紧）。
- **高频场景**：
  - 通勤 10 分钟刷 1 课 + 2 题；
  - 晚间 30 分钟走完一个单元；
  - 周末做错题回顾和小测。

## 4. 核心功能

### 4.1 卡通风格学习界面
- 移动端优先的卡片式布局；
- 吉祥物 + 高饱和色块 + 圆角按钮；
- 章节状态一眼可见（未学/进行中/已完成）。

### 4.2 账号体系
- 邮箱注册登录（V1 本地模拟，V2 接后端鉴权）；
- 个人学习档案（目标考试日、累计XP、薄弱点）。

### 4.3 学习路径与课程库
- 覆盖 CPA 六科（会计、审计、财管、税法、经济法、战略）；
- 单元 > 课时 > 题目三级结构；
- 每节课包含：目标、考点、讲解脚本、练习题、复习建议。

### 4.4 学习进度与数据闭环
- 记录课时完成状态、分数、完成时间；
- 自动识别薄弱点（错题聚合）；
- 仪表盘展示总体完成率与剩余备考天数。

### 4.5 订阅体系
- Free / Pro / Ultra 三档；
- Pro 起支持个性化 LLM 课程；
- 订阅页支持方案切换（V1 模拟支付，V2 对接支付网关）。

### 4.6 LLM 课程生成
- 输入：课时目标、考点、用户薄弱点；
- 输出：学习脚本、自适应题目、复习策略；
- V1 支持本地 fallback，V2 接真实模型网关与质检流程。

## 5. 功能边界（V1）

已包含：
- 完整前端学习闭环；
- 本地账号/进度/订阅管理；
- 课程数据种子；
- LLM 生成接口抽象。

暂不包含（V2）：
- 真实后端鉴权与支付；
- 题库审核后台；
- 教研运营后台与 AB 测试系统。

## 6. 指标设计（上线后）

- 激活率：注册后 24h 内完成首课比例；
- 日活学习率：DAU 中完成至少 1 课比例；
- 七日留存；
- 人均周学习时长；
- 题目正确率提升曲线；
- 订阅转化率（Free -> Pro/Ultra）。
